**INTRODUCTION**

So, here is the code for 8-puzzle problem using python 
and it performs the 4 major uninformed searches:
 1. Breadth First search
 2. Depth First search
 3. Uniform Cost search
 4. Iterative Deepening search

**WORKING**
IT calculates the path taken through each of the search
algos and returns
 1.time taken
 2.path Cost
 3.path taken

**INPUT CONSTRAINTS**
Please enter string inputs for initial and final 
states of puzzle 
    E.g "120345678" , "012345678"
    Where 0 is for blank space

**FILE RUNNING**
Clone the repo and run code.py file AS
    python code.py
